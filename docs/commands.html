AeroBot's Commands

Moderation

kick

aliases: kick
expected args: <user ping> [reason]
Example:
>kick @aero_noob kick lol
Note: that should be a real mention of a user.

ban

aliases: ban
expected args: <user ping/id> <duration/forever> [reason]
Example:

Tempban: >ban @aero_noob 7d 7 day tempban
Permaban: >ban @aero_noob forever permaent ban

Note: that should be a real mention of a user (or their ID)

unban

Can unban both tempbans and permabans. aliases: unban
expected args: <user ID>
Example: >unban 645161965411

mute

aliases: mute
expected args: <user ping/id> <duration/forever> [reason]
Example: Follow usage of ban (but use mute instead of ban)

unmute

Can unmute both temporary and permanent mutes. aliases: unmute
expected args: <user ping>
Example: >unmute @aero_noob Note: That should be a real mention of a user.
